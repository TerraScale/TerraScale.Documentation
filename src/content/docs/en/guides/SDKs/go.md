## Go SDK (Coming Soon)

The TerraScale Go SDK is currently in development. This page will be updated with full documentation once the SDK is available.

## Planned Features

The first Go SDK release is planned to include:

- Database operations for storing, reading, updating, and deleting items
- Query support for common access patterns
- Type-safe models for cleaner application code
- Error handling helpers for API and transport failures
- Connection management for production workloads

## Stay Updated

Want to know when the Go SDK is ready?

- Join the [Discord community](https://discord.gg/8Zr2Nw9g) for announcements
- Watch the [TerraScale GitHub organization](https://github.com/TerraScale)

## In the Meantime

You can use the TerraScale REST API directly from Go today with the standard library or your preferred HTTP client.

Start with the [API reference](/reference/api/) to learn the base URL, authentication model, and request patterns.
