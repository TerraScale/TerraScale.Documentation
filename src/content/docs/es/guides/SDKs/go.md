## SDK Go (próximamente)

El SDK Go de TerraScale está actualmente en desarrollo. Esta página se actualizará con la documentación completa cuando el SDK esté disponible.

## Funciones planificadas

Está previsto que la primera versión del SDK Go incluya:

- Operaciones de base de datos para almacenar, leer, actualizar y eliminar elementos
- Soporte de consultas para patrones de acceso comunes
- Modelos con seguridad de tipos para un código de aplicación más limpio
- Helpers de manejo de errores para fallos de la API y del transporte
- Gestión de conexiones para cargas de trabajo de producción

## Mientras tanto

Hoy ya puedes usar la REST API de TerraScale directamente desde Go con la biblioteca estándar o con tu cliente HTTP preferido.

Empieza con la [referencia de la API](/reference/api/) para conocer la URL base, el modelo de autenticación y los patrones de solicitud.
