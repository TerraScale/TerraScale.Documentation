## SDK TypeScript (próximamente)

El SDK TypeScript de TerraScale está actualmente en desarrollo. Esta página se actualizará con la documentación completa cuando el SDK esté disponible.

## Funciones planificadas

El SDK TypeScript previsto se centrará en flujos de trabajo modernos de JavaScript y TypeScript, incluidos:

- Operaciones de base de datos para los flujos principales de elementos
- Soporte de consultas para patrones de lectura comunes
- Modelos con seguridad de tipos para un mejor soporte del editor y refactorizaciones más seguras
- Paquetes tree-shakeable para mantener bundles pequeños
- Compatibilidad con Node.js y edge runtimes

## Mantente al día

Únete a la [comunidad de Discord](https://discord.gg/8Zr2Nw9g) para anuncios y actualizaciones de lanzamientos.

## Mientras tanto

Hasta que el SDK esté disponible, puedes llamar a la REST API de TerraScale directamente desde TypeScript o Node.js con `fetch` o cualquier cliente HTTP.

Consulta la [referencia de la API](/reference/api/) para empezar con endpoints, autenticación y ejemplos de solicitudes.
