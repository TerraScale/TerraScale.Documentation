## TypeScript SDK (Coming Soon)

The TerraScale TypeScript SDK is currently in development. This page will be updated with full documentation once the SDK is available.

## Planned Features

The planned TypeScript SDK will focus on modern JavaScript and TypeScript workflows, including:

- Database operations for core item workflows
- Query support for common read patterns
- Type-safe models for better editor support and safer refactoring
- Tree-shakeable packages to keep bundles small
- Compatibility with Node.js and edge runtimes

## Stay Updated

Join the [Discord community](https://discord.gg/8Zr2Nw9g) for announcements and release updates.

## In the Meantime

Until the SDK ships, you can call the TerraScale REST API directly from TypeScript or Node.js with `fetch` or any HTTP client.

See the [API reference](/reference/api/) to get started with endpoints, authentication, and request examples.
