## Nossa filosofia de código aberto

Acreditamos que o open source pode criar ferramentas melhores, comunidades mais fortes e software mais transparente. O TerraScale apoia esse modelo, e hoje já mantemos alguns repositórios públicos.

## O que é open source hoje

Estes projetos do TerraScale são open source atualmente:

- [TerraScale Dashboard](https://github.com/TerraScale/TerraScale.Dashboard)
- [TerraScale Documentation](https://github.com/TerraScale/TerraScale.Documentation)

## Por que nem tudo é open source ainda

Nem toda parte do TerraScale é open source hoje. Isso não acontece porque somos contra open source. Neste momento, nosso foco está em entregar o produto, apoiar usuários e garantir que tudo o que publicarmos esteja pronto para ser mantido com qualidade. Tornar um projeto open source exige tempo, revisão e cuidado contínuo, e queremos fazer isso com responsabilidade.

## Como isso pode evoluir

Revisamos essa política regularmente. À medida que o produto e a equipe continuam crescendo, podemos abrir mais partes do TerraScale quando isso fizer sentido para o projeto e para a comunidade.

## Contribuindo

Se você quiser contribuir, comece pelos nossos repositórios públicos no GitHub. Procure issues abertas, proponha melhorias ou envie pull requests para documentação e mudanças relacionadas ao dashboard.

- Navegue por nossos repositórios no [GitHub](https://github.com/TerraScale)
- Contribua com o [repositório de documentação](https://github.com/TerraScale/TerraScale.Documentation)
- Contribua com o [repositório do dashboard](https://github.com/TerraScale/TerraScale.Dashboard)
