## SDK Go (em breve)

O SDK Go do TerraScale está atualmente em desenvolvimento. Esta página será atualizada com a documentação completa quando o SDK estiver disponível.

## Recursos planejados

Está previsto que a primeira versão do SDK Go inclua:

- Operações de banco de dados para armazenar, ler, atualizar e excluir itens
- Suporte a consultas para padrões de acesso comuns
- Modelos com segurança de tipos para um código de aplicação mais limpo
- Helpers de tratamento de erros para falhas da API e de transporte
- Gerenciamento de conexões para cargas de trabalho de produção

## Fique por dentro

Quer saber quando o SDK Go estiver pronto?

- Entre na [comunidade no Discord](https://discord.gg/8Zr2Nw9g) para anúncios
- Acompanhe a [organização TerraScale no GitHub](https://github.com/TerraScale)

## Enquanto isso

Hoje você já pode usar a REST API do TerraScale diretamente com Go usando a biblioteca padrão ou o cliente HTTP de sua preferência.

Comece pela [referência da API](/reference/api/) para conhecer a URL base, o modelo de autenticação e os padrões de requisição.
