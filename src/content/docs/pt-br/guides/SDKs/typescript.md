## SDK TypeScript (em breve)

O SDK TypeScript do TerraScale está atualmente em desenvolvimento. Esta página será atualizada com a documentação completa quando o SDK estiver disponível.

## Recursos planejados

O SDK TypeScript planejado terá foco em fluxos de trabalho modernos de JavaScript e TypeScript, incluindo:

- Operações de banco de dados para fluxos principais de itens
- Suporte a consultas para padrões comuns de leitura
- Modelos com segurança de tipos para melhor suporte do editor e refatorações mais seguras
- Pacotes tree-shakeable para manter bundles pequenos
- Compatibilidade com Node.js e edge runtimes

## Fique por dentro

Entre na [comunidade no Discord](https://discord.gg/8Zr2Nw9g) para anúncios e atualizações de lançamento.

## Enquanto isso

Até o SDK ser lançado, você pode chamar a REST API do TerraScale diretamente de TypeScript ou Node.js com `fetch` ou qualquer cliente HTTP.

Consulte a [referência da API](/reference/api/) para começar com endpoints, autenticação e exemplos de requisição.
