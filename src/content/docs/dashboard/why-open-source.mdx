---
title: Why is TerraScale dashboard open source?
description: Why is TerraScale dashboard open source?
---

The TerraScale Dashboard is open source because it is a great example on how to use TerraScale database.

The TerraScale Dashboard is built using:
- Blazor
- Tailwind v4
- [TerraScale.SDK.CSharp](/guides/sdks/csharp)

Check the source at [TerraScale Dashboard GitHub repository](https://github.com/TerraScale/TerraScale.Dashboard)
